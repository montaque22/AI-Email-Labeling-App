# Emailable

Emailable is an AI-assisted email organization and automation application. It connects multiple email accounts, synchronizes labels or folders, classifies messages, drafts replies, and creates reviewable rules when the AI is uncertain.

It is designed for people and teams who want to automate email workflows without losing visibility or control. Emailable keeps account connections, categories, prompts, confidence settings, rules, metrics, APIs, webhooks, MCP tools, and Home Assistant support in one place.

## Why use Emailable

- Organize multiple email accounts with one consistent category system.
- Apply AI classifications while keeping uncertain decisions available for review.
- Reuse prompts and confidence settings across n8n, Home Assistant, MCP, and API workflows.
- Run through npm, Docker, Coolify, or as a Home Assistant add-on.
